Installation Svelte Kit
#################################
npm init svelte@next my-app
cd my-app
npm install
npm run dev

(PostCSS Preset Env | PostCSS Preset Env lets you convert modern CSS into something most browsers can understand)
(PostCSS Stylelint | A mighty, modern linter that helps you avoid errors and enforce conventions in your styles)
